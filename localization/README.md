# Ortalab Localization Files

This directory contains localization files for the Ortalab mod.

## Available Languages

- `default.lua` - English (default)
- `es_ES.lua` - Spanish (Spain)
- `es_419.lua` - Spanish (Latin America)
- `vi.lua` - Vietnamese
- `zh_CN.lua` - Simplified Chinese (简体中文) ✨ NEW

## Simplified Chinese (简体中文)

The `zh_CN.lua` file provides Simplified Chinese localization for the Ortalab mod.

### Coverage

- ✓ All UI interface text
- ✓ Configuration menu options
- ✓ Challenge names
- ✓ Labels and status messages
- ✓ Mod description
- ✓ Artist credits
- ✓ Common game terminology

### File Statistics

- **Lines:** 3,802
- **Size:** ~99KB
- **Chinese Characters:** 1,138 characters
- **Translation Sequences:** 315 unique translations

### Usage

The localization file will be automatically loaded by the game when the language is set to Simplified Chinese in Balatro's settings.

### Translation Notes

- All core UI elements have been translated
- Game-specific terminology uses standard poker/gaming terms in Chinese
- Formatting codes (e.g., `{C:red}`, `{V:1}`) are preserved for proper in-game display
- Joker names and detailed item descriptions may mix English and Chinese for clarity

## Contributing

To improve or extend the Simplified Chinese translation:

1. Edit `zh_CN.lua`
2. Maintain the same Lua table structure as `default.lua`
3. Preserve all formatting codes and variable placeholders
4. Test in-game to ensure proper display
